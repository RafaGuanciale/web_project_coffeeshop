# Triple Espresso da TripleTen

Esse projeto faz parte do meu aprendizado no bootcamp da TripleTen.
Aqui desenvolvi uma página completa com layout responsivo, formulário funcional e uso de iframes, elementos que estão presentes em praticamente qualquer site moderno.

Foi a primeira vez que pratiquei a criação de formulários e a incorporação de conteúdos externos usando iframes, o que representou um grande passo na minha evolução como desenvolvedor.

Tecnologias utilizadas

- HTML5
- CSS3
- Iframe
- Formulários HTML
- Metodologia BEM (para organização do CSS)

Principais aprendizados

- Estruturação de páginas utilizando boas práticas de semântica.
- Posicionamento e estilização de elementos com CSS.
- Implementação de um formulário acessível e bem formatado.
- Inserção de conteúdos externos com iframe.

Considerações finais

Me sinto muito orgulhoso do avanço que fiz nesse projeto e de todo o aprendizado obtido.
Cada sprint tem sido um passo importante para consolidar minha base em desenvolvimento web.
